MATLAB符号运算

符号对象的建立

syms可用于建立一个或多个符号对象，同时也支持使用sym创建一个符号对象

 
xxxxxxxxxx
x=sym("x")
syms y z

符号运算

求极限

limit用于求符号表达式 的极限，语法如下：

 
xxxxxxxxxx
syms x a
I1=limit('(sin(x)-sin(3*x))/sin(x)',x,0)
I1=
    -2

以上为的值的符号解法。

求导

diff用于求符号表达式的导数，语法如下：

 
xxxxxxxxxx
syms x
y=sin(x);
diff(y)
ans=
    cos(x)

求积分

int可用于求解符号积分，语法如下：

是积分变量，是积分上下界

 
xxxxxxxxxx
syms x y z
I1=int(1/(3+2*x+x^2),x,0,1)
I1=
    -(2^(1/2)*(atan(8^(1/2)/4)-atan(2^(1/2))))/2

化简、提取和代入

factor(因式分解)、expand(展开)、collect(S,v)(按v合并同类项，缺省时自动选取)

simplify(化简)、simple(调用MATLAB的其他函数进行化简，并显示过程)

subs(替换)

 
xxxxxxxxxx
syms a b c x y a0 y0
f=a*b+c/x*y;
a='a0';b=1;c=4;x='x0';y=5;
t=subs(f)
%第二种
subs(f,{a,b,c,x,y},{'a0',1,4,'x0',5})
t=
    a0+20/x0

级数

symsum可用于求符号级数，语法如下：

s为级数通项，v为求和变量，n、m为求和的开始项和结束项

 
xxxxxxxxxx
syms n
s=symsum(1/n^2,1,inf)
s=
    pi^2/6
y=symsum(1/(2*n-1),1,10)
y=
    31037876/14549535
eval(y)
ans=
    2.1333

有些版本为 ，a为初值，n为阶数

 
xxxxxxxxxx
syms x
f1=(1+x+x^2)/(1-x+x^2);
taylor(f1,x,12)     %缺省时a为0
ans=
    2*x^8 - 2*x^10 - 2*x^11 + 2*x^7 - 2*x^5 - 2*x^4 + 2*x^2 + 2*x + 1

求解方程

  • solve用于求解符号方程组

    • 语法：

    为符号方程，为符号变量

  • dsolve用于求解符号微分方程组

    • 语法：

    为符号方程，为给定的初值，为符号变量

例：

1. 

 
xxxxxxxxxx
syms x y z a
x=solve('1/(x+2)+a=1/(x-2)','x')
x=
    （2*(a^2+a)^(1/2))/a
    -（2*(a^2+a)^(1/2))/a

2. 

 
xxxxxxxxxx
y=dsolve('Dy=(t^2+y^2)/t^2/2','t')
y=
    -t*(1/(C7+log(t)/2)-1)
